# Self_Official_Web

## 介绍
Self_Official_Web 是一个个人使用的官方网站前端项目，适用于个人与官方用途。

## 软件架构
该项目基于现代前端技术栈构建，使用 Vue.js 作为主要框架，并采用 Vite 作为构建工具。整体结构分为以下主要部分：
- **前端页面**：由 Vue 组件构成，包含主页、产品管理、文件管理等模块。
- **API 接口**：封装了与后端交互的请求方法，包括认证、产品管理、文件管理等接口。
- **静态资源**：包含 CSS 样式、图片和图标等资源。
- **配置文件**：用于构建、开发和调试的配置，如 `vite.config.js` 和 `jsconfig.json`。

## 安装教程
1. 确保本地安装了 [Node.js](https://nodejs.org) 和 [Yarn](https://yarnpkg.com)。
2. 克隆仓库到本地：
   ```bash
   git clone https://gitee.com/yanke_4_0/self_-official_-web.git
   ```
3. 进入项目目录并安装依赖：
   ```bash
   cd self_-official_-web/frontend
   yarn install
   ```

## 使用说明
1. 启动本地开发服务器：
   ```bash
   yarn dev
   ```
2. 构建生产版本：
   ```bash
   yarn build
   ```
3. 预览生产环境构建结果：
   ```bash
   yarn preview
   ```
